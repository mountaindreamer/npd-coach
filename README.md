# NPD 应对教练 — AI 心理自助训练工具

通过 AI 模拟对话和教练指导，帮助 NPD（自恋型人格障碍）关系受影响者识别操控模式、练习应对策略、重建心理边界。

## 功能

### 模拟对话训练
AI 扮演具有 NPD 特征的角色，用户在安全环境中练习应对。

- 5 个预设场景（亲密关系 / 亲子关系 / 职场）
- 3 个难度级别（初级 → 中级 → 高级）
- 对话结束后获得 AI 反馈评分

### 教练指导
AI 以专业心理教练身份提供学习和指导。

- 5 个结构化学习模块（识别模式、应对策略、情绪管理、边界设定、自我重建）
- 自由对话模式
- 融合 CBT/DBT 技术

### 安全机制
- 危机关键词检测
- SOS 紧急求助按钮（心理危机热线）
- 模拟对话后强制减压环节
- 免责声明

## 快速开始

```bash
# 安装依赖
npm install

# 配置环境变量
cp .env.example .env.local
# 编辑 .env.local 填入你的 API Key

# 启动开发服务器
npm run dev
```

打开 http://localhost:3000 访问。

## 环境变量

| 变量 | 说明 | 默认值 |
|------|------|--------|
| `GOOGLE_GENERATIVE_AI_API_KEY` | Google Gemini API 密钥 | 必填 |
| `GOOGLE_MODEL` | 使用的模型名称 | `gemini-2.5-flash-preview-04-17` |

## 技术栈

- **框架**: Next.js 16 (App Router)
- **样式**: Tailwind CSS 4
- **AI**: Vercel AI SDK + Google Gemini
- **语言**: TypeScript
- **存储**: localStorage（训练记录）
